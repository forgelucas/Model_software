Realiza Agendamento

UMLUseCase
Untitled :: Model :: Realiza Agendamento

Description

Pré-condições Solicitante escolhe o(s) produto(s) desejado(s)

Pós-condições Solicitante confirma a data escolhida

Fluxo Principal

1 - Solicitante escolhe o(s) produto(s) desejado(s) 2 - Solicitante opta por realizar o pedido 3 - Sistema exibe página com os itens escolhidos e solicita o tipo de serviço 4 - Solicitante escolhe o tipo de serviço e informa os dados necessários 5 - Sistema exibe página com os itens escolhidos, tipo de serviço escolhido e solicita o agendamento do serviço 6 - Solicitante escolhe a data desejada para o agendamento 7 - Sistema confirma a data escolhida 8 - Solicitante confirma agendamento Fluxo Alternativo:

6 - Solicitante escolhe uma data indisponível para o agendamento 6.1 - Sistema informa que a data está indisponível e solicita outra data 6.2 - Solicitante escolhe uma data que esteja disponível para o agendamento 6.3 - Sistema informa que a data está disponível 6.4 - Solicitante confirma a data de agendamento

Fluxo alternativo:

6 - Solicitante escolhe uma data indisponível para o agendamento 6.1 - Tireguard informa que a data está indisponível e solicita outra data 6.2 - Solicitante escolhe uma data que esteja disponível para o agendamento 6.3 - Tireguard informa que a data está disponível 6.4 - Solicitante confirma a data de agendamento
