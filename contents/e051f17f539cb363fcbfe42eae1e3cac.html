Realiza Pagamento

UMLUseCase
Untitled :: Model :: Realiza Pagamento

Description

Pré-condições Solicitante deve ter realizado o agendamento

Pós-condições Solicitante realiza o pagamento com sucesso

Fluxo Principal

1 - TireGuard exibe as opções de pagamento para o solicitante 2 - Solicitante escolhe a opção de pagamento 3 - Solicitante confirma a forma de pagamento 4 - TireGuard confirma os dados 5 - Solicitante realiza o pagamento

Fluxo Alternativo:

2.1 - Solicitante opta por cadastrar um novo cartão 2.2 - Sistema solicita dados necessários 2.3 - Solicitante informa os dados 2.4 - Sistema confirma os dados 2.5 - Solicitante realiza o pagamento

Fluxo de Exceção

1 - Solicitante informa algum dado errado na hora de efetuar o pagamento 2 - Sistema informa ao solicitante que os dados foram inseridos incorretamente 3 - Solicitante revisa e corrige os dados errados 4 - Sistema confirma os dados 5 - Solicitante realiza o pagamento
